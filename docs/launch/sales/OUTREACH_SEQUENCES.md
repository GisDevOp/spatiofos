## Sequence A — Homeowner (post-storm)

**Day 0 — Email**

Subject: Quick roof check without anyone on the roof — Brisbane / SEQ

Body highlights: drone capture, AI-assisted report, 24hr target turnaround, Essential $199 launch.

**Day 3 — Follow-up**

Subject: Re: roof documentation after recent weather

Offer free Roof Score registration (when live) or Essential inspection.

**Day 7 — Break-up**

Subject: Closing the loop

One-line CTA to book or reply "not now".

## Discovery questions (`@sales-outreach` / `@offer-lead-gen-strategist`)

1. How many properties / roofs in portfolio?
2. Last roof inspection method and pain points?
3. Insurer or AGM documentation requirements?
4. Timeline for next inspection cycle?
5. Decision maker for procurement?

## Objection handlers

| Objection | Response |
|-----------|----------|
| "Insurer won't accept drone" | Premium report is insurance-structured; client should confirm with provider |
| "We use local roofers" | Complement periodic physical access with baseline drone documentation |
| "Too expensive" | Launch Essential $199 vs standard $399; no roof access risk |
